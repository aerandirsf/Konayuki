# Konayuki

[English](./README.md)

Konayuki 是一套 Typora 主题，包含两份文件：`konayuki-light.css` 与 `konayuki-dark.css`，主打清爽、可读性强的排版与配色。

## 使用方法

1. Typora：`设置` → `外观` → `打开主题文件夹`
2. 将 `konayuki-light.css` 和 `konayuki-dark.css` 复制到主题目录
3. 退出 Typora 并重新打开
4. 菜单栏 `主题` 选择：
   - `Konayuki Light`
   - `Konayuki Dark`

## 自定义

- 字体：在 `konayuki-light.css` / `konayuki-dark.css` 顶部修改 `--font-ui` / `--font-mono`
- 平台说明：`--font-ui` 使用系统字体栈，不同系统显示的默认 UI 字体可能不同（尤其是 Windows）。
- 代码字体：默认等宽字体来自 [Maple Font](https://github.com/subframe7536/maple-font)（`Maple Mono NF CN`）。建议安装该字体以获得最佳效果；否则会回退到 `--font-mono` 中的后续字体。
